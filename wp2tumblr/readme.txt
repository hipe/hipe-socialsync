Send your posts automatically on Tumblr.

== Description ==

Wp2Tumblr adds a checkbox to WRITE POST editor. It allows you when publish your post to post-it automatically on your Tumblr blog, if it is checked. Support for cURL is required. 

== Installation ==

1. Upload `wp2tumblr.php` to the `/wp-content/plugins/` directory
2. Active the plugin through the 'Plugins' menu in WordPress
